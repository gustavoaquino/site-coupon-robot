1. Qual a veracidade do site?

O site é fruto de um desenvolvedor independente, aka Aquino. Cujo objetivo é disponibilizar cupons de desconto de forma automática.

2. Dê onde sai os cupons?

Todos os cupons são capturados através de uma integração disponibilizada por uma plataforma, essa plataforma é alimentada com os cupons diretamente pelas lojas, e o trabalho desse site é agrupar as lojas e os seus cupons.
